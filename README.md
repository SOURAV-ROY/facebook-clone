 # Facebook Clone #
 
 
## Use the package manager [npm](https://www.npmjs.com/) to install
```
npm install @material-ui/core
npm install @material-ui/icons
npm install firebase
```
## Firebase Hosting
```
npm install -g firebase-tools
firebase logout
firebase login
firebase init
firebase deploy
```
